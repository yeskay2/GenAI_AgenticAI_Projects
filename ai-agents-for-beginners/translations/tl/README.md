> Kasama sa repositoryong ito ang higit sa 50 wika na mga pagsasalin na makabuluhang nagpapalaki ng laki ng pag-download. Para mag-clone nang walang mga pagsasalin, gamitin ang sparse checkout:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> Binibigyan ka nito ng lahat ng kailangan mo para matapos ang kurso nang mas mabilis ang pag-download.

## üå± Pagsisimula

Ang kursong ito ay may mga leksyon na sumasaklaw sa mga batayan ng paggawa ng AI Agents. Bawat leksyon ay may sariling paksa kaya simulan mo kung saan mo gusto!

May suporta sa maraming wika para sa kursong ito. Pumunta sa aming [mga available na wika dito](../..).

Kung ito ang iyong unang pagkakataon na gumawa gamit ang Generative AI models, tingnan ang aming [Generative AI For Beginners](https://aka.ms/genai-beginners) na kurso, na may 21 na leksyon tungkol sa paggawa gamit ang GenAI.

Huwag kalimutang [i-star (üåü) ang repo na ito](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) at [i-fork ang repo na ito](https://github.com/microsoft/ai-agents-for-beginners/fork) para patakbuhin ang code.

### Makipagkilala sa Ibang Nag-aaral, Sagutin ang Iyong Mga Tanong

Kung ikaw ay nahihirapan o may mga tanong tungkol sa paggawa ng AI Agents, sumali sa aming dedikadong Discord Channel sa [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Ano ang Kailangan Mo

Bawat leksyon sa kursong ito ay may kasamang mga halimbawa ng code, na matatagpuan sa folder na code_samples. Maaari mong [i-fork ang repo na ito](https://github.com/microsoft/ai-agents-for-beginners/fork) para gumawa ng sarili mong kopya.

Ang mga halimbawa ng code sa mga gawaing ito ay gumagamit ng Azure AI Foundry at GitHub Model Catalogs para sa pakikipag-interact sa Language Models:

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - Libre / Limitado
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Kailangan ng Azure Account

Ginagamit din sa kursong ito ang mga sumusunod na AI Agent frameworks at mga serbisyo mula sa Microsoft:

- [Microsoft Agent Framework (MAF) - Bago!](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)

Para sa karagdagang impormasyon tungkol sa pagpapatakbo ng code para sa kursong ito, pumunta sa [Course Setup](./00-course-setup/README.md).
