**Agents wey dey for customer support process**:

- **Customer agent**: Na dis agent dey represent di customer, e dey responsible to start di support process.
- **Support agent**: Dis agent dey represent di support process, e dey responsible to help di customer.
- **Escalation agent**: Dis agent dey represent di escalation process, e dey responsible to carry di matter go higher level of support.
- **Resolution agent**: Dis agent dey represent di resolution process, e dey responsible to solve any wahala wey fit happen for di support process.
- **Feedback agent**: Dis agent dey represent di feedback process, e dey responsible to collect feedback from di customer.
- **Notification agent**: Dis agent dey represent di notification process, e dey responsible to send notification give di customer for different stage of di support process.
- **Analytics agent**: Dis agent dey represent di analytics process, e dey responsible to check data wey concern di support process.
- **Audit agent**: Dis agent dey represent di audit process, e dey responsible to check di support process to make sure say dem dey do am well.
- **Reporting agent**: Dis agent dey represent di reporting process, e dey responsible to create report about di support process.
- **Knowledge agent**: Dis agent dey represent di knowledge process, e dey responsible to keep knowledge base wey get information about di support process.
- **Security agent**: Dis agent dey represent di security process, e dey responsible to make sure say di support process dey secure.
- **Quality agent**: Dis agent dey represent di quality process, e dey responsible to make sure say di support process dey of better quality.
- **Compliance agent**: Dis agent dey represent di compliance process, e dey responsible to make sure say di support process follow di rules and policy.
- **Training agent**: Dis agent dey represent di training process, e dey responsible to train support agents on how dem go fit help customers.

E plenty small, abi e pass wetin you dey expect?
