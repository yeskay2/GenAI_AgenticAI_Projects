# AGENTS.md

## ಯೋಜನೆಯ ಅವಲೋಕನ

ಈ ರೆಪೊಸಿಟರಿಯಲ್ಲಿ "AI ಏಜೆಂಟ್ಸ್ ಫಾರ್ ಬಿಗಿನ್ನರ್ಸ್" ಎಂಬ ಶೈಕ್ಷಣಿಕ ಕೋರ್ಸ್ ಅನ್ನು ಒಳಗೊಂಡಿದೆ - AI ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಿಸಲು ಅಗತ್ಯವಿರುವ ಎಲ್ಲವನ್ನು ಕಲಿಸುವ ಸಮಗ್ರ ಕೋರ್ಸ್. ಈ ಕೋರ್ಸ್ 15+ ಪಾಠಗಳನ್ನು ಒಳಗೊಂಡಿದ್ದು, ಮೂಲಭೂತಗಳು, ವಿನ್ಯಾಸ ಮಾದರಿಗಳು, ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳು ಮತ್ತು AI ಏಜೆಂಟ್ಸ್‌ನ ಉತ್ಪಾದನಾ ನಿಯೋಜನೆಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.

**ಮುಖ್ಯ ತಂತ್ರಜ್ಞಾನಗಳು:**
- Python 3.12+
- ಇಂಟರಾಕ್ಟಿವ್ ಕಲಿಕೆಗೆ Jupyter Notebooks
- AI ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳು: Semantic Kernel, AutoGen, Microsoft Agent Framework (MAF)
- Azure AI ಸೇವೆಗಳು: Azure AI Foundry, Azure AI Agent Service
- GitHub Models Marketplace (ಉಚಿತ ಟಿಯರ್ ಲಭ್ಯವಿದೆ)

**ಆರ್ಕಿಟೆಕ್ಚರ್:**
- ಪಾಠ ಆಧಾರಿತ ರಚನೆ (00-15+ ಡೈರೆಕ್ಟರಿಗಳು)
- ಪ್ರತಿ ಪಾಠದಲ್ಲಿ: README ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಕೋಡ್ ಮಾದರಿಗಳು (Jupyter notebooks), ಮತ್ತು ಚಿತ್ರಗಳು
- ಸ್ವಯಂಚಾಲಿತ ಅನುವಾದ ವ್ಯವಸ್ಥೆಯ ಮೂಲಕ ಬಹುಭಾಷಾ ಬೆಂಬಲ
- ಪ್ರತಿ ಪಾಠಕ್ಕೆ ಬಹು ಫ್ರೇಮ್‌ವರ್ಕ್ ಆಯ್ಕೆಗಳು (Semantic Kernel, AutoGen, Azure AI Agent Service)

## ಸೆಟಪ್ ಕಮಾಂಡ್‌ಗಳು

### ಪೂರ್ವಾಪೇಕ್ಷಿತಗಳು
- Python 3.12 ಅಥವಾ ಹೆಚ್ಚಿನ ಆವೃತ್ತಿ
- GitHub ಖಾತೆ (GitHub Models - ಉಚಿತ ಟಿಯರ್‌ಗಾಗಿ)
- Azure ಚಂದಾದಾರಿಕೆ (ಐಚ್ಛಿಕ, Azure AI ಸೇವೆಗಳಿಗೆ)

### ಪ್ರಾಥಮಿಕ ಸೆಟಪ್

1. **ರೆಪೊಸಿಟರಿಯನ್ನು ಕ್ಲೋನ್ ಅಥವಾ ಫೋರ್ಕ್ ಮಾಡಿ:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ಅಥವಾ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python ವರ್ಚುವಲ್ ಎನ್ವಿರಾನ್‌ಮೆಂಟ್ ರಚಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಿ:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ವಿಂಡೋಸ್‌ನಲ್ಲಿ: venv\Scripts\activate
   ```

3. **ಅವಶ್ಯಕತೆಗಳನ್ನು ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿ:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ಪರಿಸರ ವ್ಯತ್ಯಾಸಗಳನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡಿ:**
   ```bash
   cp .env.example .env
   # ನಿಮ್ಮ API ಕೀಗಳು ಮತ್ತು ಎಂಡ್‌ಪಾಯಿಂಟ್‌ಗಳೊಂದಿಗೆ .env ಅನ್ನು ಸಂಪಾದಿಸಿ
   ```

### ಅಗತ್ಯವಿರುವ ಪರಿಸರ ವ್ಯತ್ಯಾಸಗಳು

**GitHub Models (ಉಚಿತ)**ಗಾಗಿ:
- `GITHUB_TOKEN` - GitHub ನಿಂದ ವೈಯಕ್ತಿಕ ಪ್ರವೇಶ ಟೋಕನ್

**Azure AI ಸೇವೆಗಳಿಗೆ** (ಐಚ್ಛಿಕ):
- `PROJECT_ENDPOINT` - Azure AI Foundry ಪ್ರಾಜೆಕ್ಟ್ ಎಂಡ್‌ಪಾಯಿಂಟ್
- `AZURE_OPENAI_API_KEY` - Azure OpenAI API ಕೀ
- `AZURE_OPENAI_ENDPOINT` - Azure OpenAI ಎಂಡ್‌ಪಾಯಿಂಟ್ URL
- `AZURE_OPENAI_CHAT_DEPLOYMENT_NAME` - ಚಾಟ್ ಮಾದರಿಗಾಗಿ ನಿಯೋಜನೆ ಹೆಸರು
- `AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME` - ಎಂಬೆಡಿಂಗ್‌ಗಳಿಗಾಗಿ ನಿಯೋಜನೆ ಹೆಸರು
- `.env.example` ನಲ್ಲಿ ತೋರಿಸಿದಂತೆ ಹೆಚ್ಚುವರಿ Azure ಸಂರಚನೆ

## ಅಭಿವೃದ್ಧಿ ಕಾರ್ಯಪ್ರವಾಹ

### Jupyter Notebooks ಅನ್ನು ಚಲಾಯಿಸುವುದು

ಪ್ರತಿ ಪಾಠವು ವಿಭಿನ್ನ ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳಿಗೆ ಅನೇಕ Jupyter notebooks ಅನ್ನು ಒಳಗೊಂಡಿದೆ:

1. **Jupyter ಪ್ರಾರಂಭಿಸಿ:**
   ```bash
   jupyter notebook
   ```

2. **ಒಂದು ಪಾಠ ಡೈರೆಕ್ಟರಿಯ ಕಡೆಗೆ ಹೋಗಿ** (ಉದಾ., `01-intro-to-ai-agents/code_samples/`)

3. **Notebooks ತೆರೆಯಿರಿ ಮತ್ತು ಚಲಾಯಿಸಿ:**
   - `*-semantic-kernel.ipynb` - Semantic Kernel ಫ್ರೇಮ್‌ವರ್ಕ್ ಬಳಸಿ
   - `*-autogen.ipynb` - AutoGen ಫ್ರೇಮ್‌ವರ್ಕ್ ಬಳಸಿ
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) ಬಳಸಿ
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) ಬಳಸಿ
   - `*-azureaiagent.ipynb` - Azure AI Agent Service ಬಳಸಿ

### ವಿಭಿನ್ನ ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುವುದು

**Semantic Kernel + GitHub Models:**
- GitHub ಖಾತೆಯೊಂದಿಗೆ ಉಚಿತ ಟಿಯರ್ ಲಭ್ಯವಿದೆ
- ಕಲಿಕೆ ಮತ್ತು ಪ್ರಯೋಗಗಳಿಗೆ ಉತ್ತಮ
- ಫೈಲ್ ಮಾದರಿ: `*-semantic-kernel*.ipynb`

**AutoGen + GitHub Models:**
- GitHub ಖಾತೆಯೊಂದಿಗೆ ಉಚಿತ ಟಿಯರ್ ಲಭ್ಯವಿದೆ
- ಬಹು-ಏಜೆಂಟ್ ಸಂಯೋಜನೆ ಸಾಮರ್ಥ್ಯಗಳು
- ಫೈಲ್ ಮಾದರಿ: `*-autogen.ipynb`

**Microsoft Agent Framework (MAF):**
- Microsoft ನಿಂದ ಹೊಸ ಫ್ರೇಮ್‌ವರ್ಕ್
- Python ಮತ್ತು .NET ನಲ್ಲಿ ಲಭ್ಯವಿದೆ
- ಫೈಲ್ ಮಾದರಿ: `*-agent-framework.ipynb`

**Azure AI Agent Service:**
- Azure ಚಂದಾದಾರಿಕೆ ಅಗತ್ಯವಿದೆ
- ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ವೈಶಿಷ್ಟ್ಯಗಳು
- ಫೈಲ್ ಮಾದರಿ: `*-azureaiagent.ipynb`

## ಪರೀಕ್ಷಾ ಸೂಚನೆಗಳು

ಇದು ಶೈಕ್ಷಣಿಕ ರೆಪೊಸಿಟರಿ, ಉದಾಹರಣೆ ಕೋಡ್‌ಗಳೊಂದಿಗೆ, ಸ್ವಯಂಚಾಲಿತ ಪರೀಕ್ಷೆಗಳಿಲ್ಲದ ಉತ್ಪಾದನಾ ಕೋಡ್ ಅಲ್ಲ. ನಿಮ್ಮ ಸೆಟಪ್ ಮತ್ತು ಬದಲಾವಣೆಗಳನ್ನು ಪರಿಶೀಲಿಸಲು:

### ಕೈಯಾರೆ ಪರೀಕ್ಷೆ

1. **Python ಪರಿಸರವನ್ನು ಪರೀಕ್ಷಿಸಿ:**
   ```bash
   python --version  # 3.12+ ಆಗಿರಬೇಕು
   pip list | grep -E "(autogen|semantic-kernel|azure-ai)"
   ```

2. **Notebook ಕಾರ್ಯಗತಗೊಳಿಸುವಿಕೆಯನ್ನು ಪರೀಕ್ಷಿಸಿ:**
   ```bash
   # ನೋಟ್ಬುಕ್ ಅನ್ನು ಸ್ಕ್ರಿಪ್ಟ್‌ಗೆ ಪರಿವರ್ತಿಸಿ ಮತ್ತು ಚಲಾಯಿಸಿ (ಆಮದುಗಳನ್ನು ಪರೀಕ್ಷಿಸುತ್ತದೆ)
   jupyter nbconvert --to script <lesson-folder>/code_samples/<notebook>.ipynb --stdout | python
   ```

3. **ಪರಿಸರ ವ್ಯತ್ಯಾಸಗಳನ್ನು ಪರಿಶೀಲಿಸಿ:**
   ```bash
   python -c "import os; from dotenv import load_dotenv; load_dotenv(); print('✓ GITHUB_TOKEN' if os.getenv('GITHUB_TOKEN') else '✗ GITHUB_TOKEN missing')"
   ```

### ವೈಯಕ್ತಿಕ Notebooks ಚಲಾಯಿಸುವುದು

Jupyter ನಲ್ಲಿ notebooks ತೆರೆಯಿರಿ ಮತ್ತು ಸೆಲ್‌ಗಳನ್ನು ಕ್ರಮವಾಗಿ ಕಾರ್ಯಗತಗೊಳಿಸಿ. ಪ್ರತಿ notebook ಸ್ವತಂತ್ರವಾಗಿದ್ದು, ಒಳಗೊಂಡಿದೆ:
- ಇಂಪೋರ್ಟ್ ಸ್ಟೇಟ್ಮೆಂಟ್‌ಗಳು
- ಸಂರಚನಾ ಲೋಡಿಂಗ್
- ಉದಾಹರಣೆ ಏಜೆಂಟ್ ಅನುಷ್ಠಾನಗಳು
- markdown ಸೆಲ್‌ಗಳಲ್ಲಿ ನಿರೀಕ್ಷಿತ ಔಟ್‌ಪುಟ್‌ಗಳು
