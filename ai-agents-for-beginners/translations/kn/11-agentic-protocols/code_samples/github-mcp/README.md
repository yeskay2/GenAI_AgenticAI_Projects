# Github MCP Server ಉದಾಹರಣೆ

## ವಿವರಣೆ

ಈ ಡೆಮೋವನ್ನು ಮೈಕ್ರೋಸಾಫ್ಟ್ ರಿಯಾಕ್ಟರ್ ಮೂಲಕ ಆಯೋಜಿಸಲಾದ AI ಏಜೆಂಟ್ಸ್ ಹ್ಯಾಕಥಾನ್‌ಗಾಗಿ ರಚಿಸಲಾಗಿದೆ.

ಈ ಸಾಧನವನ್ನು ಬಳಕೆದಾರರ Github ರೆಪೊಗಳ ಆಧಾರದ ಮೇಲೆ ಹ್ಯಾಕಥಾನ್ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು ಶಿಫಾರಸು ಮಾಡಲು ಬಳಸಲಾಗುತ್ತದೆ. ಇದು ಈ ಕೆಳಗಿನ ರೀತಿಯಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ:

1. **Github ಏಜೆಂಟ್** - Github MCP ಸರ್ವರ್ ಬಳಸಿ ರೆಪೊಗಳು ಮತ್ತು ಅವುಗಳ ಬಗ್ಗೆ ಮಾಹಿತಿಯನ್ನು ಪಡೆಯುವುದು.
2. **ಹ್ಯಾಕಥಾನ್ ಏಜೆಂಟ್** - Github ಏಜೆಂಟ್‌ನ ಡೇಟಾವನ್ನು ತೆಗೆದು, ಬಳಕೆದಾರನ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳು, ಬಳಸಿದ ಭಾಷೆಗಳು ಮತ್ತು AI ಏಜೆಂಟ್ಸ್ ಹ್ಯಾಕಥಾನ್‌ನ ಪ್ರಾಜೆಕ್ಟ್ ಟ್ರ್ಯಾಕ್‌ಗಳ ಆಧಾರದ ಮೇಲೆ ಸೃಜನಶೀಲ ಹ್ಯಾಕಥಾನ್ ಪ್ರಾಜೆಕ್ಟ್ ಐಡಿಯಾಗಳನ್ನು ನೀಡುವುದು.
3. **ಈವೆಂಟ್ಸ್ ಏಜೆಂಟ್** - ಹ್ಯಾಕಥಾನ್ ಏಜೆಂಟ್‌ನ ಶಿಫಾರಸಿನ ಆಧಾರದ ಮೇಲೆ, AI ಏಜೆಂಟ್ಸ್ ಹ್ಯಾಕಥಾನ್ ಸರಣಿಯ ಸಂಬಂಧಿತ ಈವೆಂಟ್‌ಗಳನ್ನು ಶಿಫಾರಸು ಮಾಡುವುದು.

## ಕೋಡ್ ಚಲಾಯಿಸುವುದು

### ಪರಿಸರ ಚರಾಂಶಗಳು

ಈ ಡೆಮೋ Azure Open AI Service, Semantic Kernel, Github MCP Server ಮತ್ತು Azure AI Search ಅನ್ನು ಬಳಸುತ್ತದೆ.

ಈ ಸಾಧನಗಳನ್ನು ಬಳಸಲು ನೀವು ಸರಿಯಾದ ಪರಿಸರ ಚರಾಂಶಗಳನ್ನು ಹೊಂದಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ:

```python
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=""
AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME=""
AZURE_OPENAI_ENDPOINT=""
AZURE_OPENAI_API_KEY=""
AZURE_OPENAI_API_VERSION=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## ಚೈನ್‌ಲಿಟ್ ಸರ್ವರ್ ಚಲಾಯಿಸುವುದು

MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಲು, ಈ ಡೆಮೋ ಚಾಟ್ ಇಂಟರ್ಫೇಸ್ ಆಗಿ ಚೈನ್‌ಲಿಟ್ ಅನ್ನು ಬಳಸುತ್ತದೆ.

ಸರ್ವರ್ ಅನ್ನು ಚಲಾಯಿಸಲು, ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯನ್ನು ಬಳಸಿ:

```bash
chainlit run app.py -w
```


ಇದು ನಿಮ್ಮ ಚೈನ್‌ಲಿಟ್ ಸರ್ವರ್ ಅನ್ನು `localhost:8000` ನಲ್ಲಿ ಪ್ರಾರಂಭಿಸುತ್ತದೆ ಮತ್ತು `event-descriptions.md` ವಿಷಯವನ್ನು ನಿಮ್ಮ Azure AI Search Index ನಲ್ಲಿ ತುಂಬುತ್ತದೆ.

## MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸುವುದು

Github MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಲು, "Type your message here.." ಚಾಟ್ ಬಾಕ್ಸ್ ಕೆಳಗಿನ "plug" ಐಕಾನ್ ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ:

![MCP Connect](../../../../../translated_images/kn/mcp-chainlit-1.7ed66d648e3cfb28.webp)

ಅಲ್ಲಿ ನೀವು "Connect an MCP" ಕ್ಲಿಕ್ ಮಾಡಿ Github MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಲು ಆಜ್ಞೆಯನ್ನು ಸೇರಿಸಬಹುದು:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```


"[YOUR PERSONAL ACCESS TOKEN]" ಅನ್ನು ನಿಮ್ಮ ನಿಜವಾದ Personal Access Token ಮೂಲಕ ಬದಲಾಯಿಸಿ.

ಸಂಪರ್ಕಿಸಿದ ನಂತರ, plug ಐಕಾನ್ ಹತ್ತಿರ (1) ಅನ್ನು ನೀವು ನೋಡಬಹುದು, ಇದು ಸಂಪರ್ಕವನ್ನು ದೃಢೀಕರಿಸುತ್ತದೆ. ಇಲ್ಲದಿದ್ದರೆ, `chainlit run app.py -w` ಬಳಸಿ ಚೈನ್‌ಲಿಟ್ ಸರ್ವರ್ ಅನ್ನು ಮರುಪ್ರಾರಂಭಿಸಲು ಪ್ರಯತ್ನಿಸಿ.

## ಡೆಮೋ ಬಳಸುವುದು

ಹ್ಯಾಕಥಾನ್ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು ಶಿಫಾರಸು ಮಾಡುವ ಏಜೆಂಟ್ ವರ್ಕ್‌ಫ್ಲೋ ಪ್ರಾರಂಭಿಸಲು, ನೀವು ಈ ರೀತಿಯ ಸಂದೇಶವನ್ನು ಟೈಪ್ ಮಾಡಬಹುದು:

"Github ಬಳಕೆದಾರ koreyspace ಗೆ ಹ್ಯಾಕಥಾನ್ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು ಶಿಫಾರಸು ಮಾಡಿ"

ರೌಟರ್ ಏಜೆಂಟ್ ನಿಮ್ಮ ವಿನಂತಿಯನ್ನು ವಿಶ್ಲೇಷಿಸಿ, ಯಾವ ಏಜೆಂಟ್‌ಗಳ (GitHub, Hackathon, ಮತ್ತು Events) ಸಂಯೋಜನೆ ನಿಮ್ಮ ಪ್ರಶ್ನೆಯನ್ನು ನಿರ್ವಹಿಸಲು ಸೂಕ್ತವಾಗಿದೆ ಎಂದು ನಿರ್ಧರಿಸುತ್ತದೆ. Github ರೆಪೊ ವಿಶ್ಲೇಷಣೆ, ಪ್ರಾಜೆಕ್ಟ್ ಐಡಿಯೇಶನ್ ಮತ್ತು ಸಂಬಂಧಿತ ತಂತ್ರಜ್ಞಾನ ಈವೆಂಟ್‌ಗಳ ಆಧಾರದ ಮೇಲೆ ಏಜೆಂಟ್‌ಗಳು ಸಮಗ್ರ ಶಿಫಾರಸುಗಳನ್ನು ಒದಗಿಸಲು ಒಟ್ಟಿಗೆ ಕೆಲಸ ಮಾಡುತ್ತವೆ.
