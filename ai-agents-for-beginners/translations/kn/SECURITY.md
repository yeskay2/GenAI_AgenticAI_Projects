# ಭದ್ರತೆ

ಮೈಕ್ರೋಸಾಫ್ಟ್ ನಮ್ಮ ಸಾಫ್ಟ್‌ವೇರ್ ಉತ್ಪನ್ನಗಳು ಮತ್ತು ಸೇವೆಗಳ ಭದ್ರತೆಯನ್ನು ಗಂಭೀರವಾಗಿ ಪರಿಗಣಿಸುತ್ತದೆ, ಇದರಲ್ಲಿ ನಮ್ಮ GitHub ಸಂಸ್ಥೆಗಳ ಮೂಲಕ ನಿರ್ವಹಿಸಲ್ಪಡುವ ಎಲ್ಲಾ ಮೂಲ ಕೋಡ್ ರೆಪೊಸಿಟರಿಗಳು ಸೇರಿವೆ, ಉದಾಹರಣೆಗೆ [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) ಮತ್ತು [Xamarin](https://github.com/xamarin).

ನೀವು [ಮೈಕ್ರೋಸಾಫ್ಟ್‌ನ ಭದ್ರತಾ ಅಪಾಯದ ವ್ಯಾಖ್ಯಾನ](https://aka.ms/security.md/definition)ಕ್ಕೆ ಹೊಂದುವ ಯಾವುದೇ ಮೈಕ್ರೋಸಾಫ್ಟ್-ಸ್ವಾಮ್ಯದ ರೆಪೊಸಿಟರಿಯಲ್ಲಿ ಭದ್ರತಾ ಅಪಾಯವನ್ನು ಕಂಡುಹಿಡಿದಿದ್ದರೆ, ದಯವಿಟ್ಟು ಕೆಳಗಿನಂತೆ ನಮಗೆ ವರದಿ ಮಾಡಿ.

## ಭದ್ರತಾ ಸಮಸ್ಯೆಗಳನ್ನು ವರದಿ ಮಾಡುವುದು

**ದಯವಿಟ್ಟು ಸಾರ್ವಜನಿಕ GitHub ಸಮಸ್ಯೆಗಳ ಮೂಲಕ ಭದ್ರತಾ ಅಪಾಯಗಳನ್ನು ವರದಿ ಮಾಡಬೇಡಿ.**

ಬದಲಿಗೆ, ದಯವಿಟ್ಟು ಅವುಗಳನ್ನು ಮೈಕ್ರೋಸಾಫ್ಟ್ ಸೆಕ್ಯುರಿಟಿ ರೆಸ್ಪಾನ್ಸ್ ಸೆಂಟರ್ (MSRC) ಗೆ [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report) ನಲ್ಲಿ ವರದಿ ಮಾಡಿ.

ನೀವು ಲಾಗಿನ್ ಮಾಡದೆ ಸಲ್ಲಿಸಲು ಇಚ್ಛಿಸಿದರೆ, [secure@microsoft.com](mailto:secure@microsoft.com) ಗೆ ಇಮೇಲ್ ಕಳುಹಿಸಿ. ಸಾಧ್ಯವಾದರೆ, ನಮ್ಮ PGP ಕೀ ಬಳಸಿ ನಿಮ್ಮ ಸಂದೇಶವನ್ನು ಎನ್‌ಕ್ರಿಪ್ಟ್ ಮಾಡಿ; ದಯವಿಟ್ಟು ಅದನ್ನು [ಮೈಕ್ರೋಸಾಫ್ಟ್ ಸೆಕ್ಯುರಿಟಿ ರೆಸ್ಪಾನ್ಸ್ ಸೆಂಟರ್ PGP ಕೀ ಪುಟ](https://aka.ms/security.md/msrc/pgp) ನಿಂದ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ.

ನೀವು 24 ಗಂಟೆಗಳ ಒಳಗೆ ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ಸ್ವೀಕರಿಸಬೇಕು. ಏನಾದರೂ ಕಾರಣದಿಂದ ನೀವು ಸ್ವೀಕರಿಸದಿದ್ದರೆ, ದಯವಿಟ್ಟು ನಿಮ್ಮ ಮೂಲ ಸಂದೇಶವನ್ನು ನಾವು ಸ್ವೀಕರಿಸಿದ್ದೇವೆ ಎಂಬುದನ್ನು ಖಚಿತಪಡಿಸಲು ಇಮೇಲ್ ಮೂಲಕ ಅನುಸರಿಸಿ. ಹೆಚ್ಚಿನ ಮಾಹಿತಿಯನ್ನು [microsoft.com/msrc](https://www.microsoft.com/msrc) ನಲ್ಲಿ ಕಾಣಬಹುದು.

ದಯವಿಟ್ಟು ಕೆಳಗಿನ ಮಾಹಿತಿಯನ್ನು (ನೀವು ಒದಗಿಸಬಹುದಾದಷ್ಟು) ಸೇರಿಸಿ, ಸಾಧ್ಯವಾದಷ್ಟು ಬೇಗ ನಮ್ಮ ವರದಿಯನ್ನು ವಿಶ್ಲೇಷಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ:

* ಸಮಸ್ಯೆಯ ಪ್ರಕಾರ (ಉದಾ. ಬಫರ್ ಓವರ್‌ಫ್ಲೋ, SQL ಇಂಜೆಕ್ಷನ್, ಕ್ರಾಸ್-ಸೈಟ್ ಸ್ಕ್ರಿಪ್ಟಿಂಗ್, ಇತ್ಯಾದಿ)
* ಸಮಸ್ಯೆಯ ಪ್ರಭಾವವನ್ನು ಹೊಂದಿರುವ ಮೂಲ ಕೋಡ್‌ನ ಸಂಪೂರ್ಣ ಪಾತ್‌ಗಳು
* ಪ್ರಭಾವಿತ ಮೂಲ ಕೋಡ್‌ನ ಸ್ಥಳ (ಟ್ಯಾಗ್/ಶಾಖೆ/ಕಮಿಟ್ ಅಥವಾ ನೇರ URL)
* ಸಮಸ್ಯೆಯನ್ನು ಪುನರಾವರ್ತಿಸಲು ಅಗತ್ಯವಿರುವ ಯಾವುದೇ ವಿಶೇಷ ಸಂರಚನೆ
* ಸಮಸ್ಯೆಯನ್ನು ಪುನರಾವರ್ತಿಸಲು ಹಂತ-ಹಂತದ ಸೂಚನೆಗಳು
* ಪ್ರೂಫ್-ಆಫ್-ಕಾನ್ಸೆಪ್ಟ್ ಅಥವಾ ಎಕ್ಸ್‌ಪ್ಲಾಯಿಟ್ ಕೋಡ್ (ಸಾಧ್ಯವಾದರೆ)
* ಸಮಸ್ಯೆಯ ಪರಿಣಾಮ, ಹಾನಿಕಾರಕ ವ್ಯಕ್ತಿ ಸಮಸ್ಯೆಯನ್ನು ಹೇಗೆ ದುರುಪಯೋಗಪಡಿಸಿಕೊಳ್ಳಬಹುದು ಎಂಬುದನ್ನು ಒಳಗೊಂಡಂತೆ

ಈ ಮಾಹಿತಿಯು ನಿಮ್ಮ ವರದಿಯನ್ನು ವೇಗವಾಗಿ ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ನೀವು ಬಗ್ ಬೌಂಟಿಗಾಗಿ ವರದಿ ಮಾಡುತ್ತಿದ್ದರೆ, ಹೆಚ್ಚು ಸಂಪೂರ್ಣ ವರದಿಗಳು ಹೆಚ್ಚಿನ ಬೌಂಟಿ ಬಹುಮಾನಕ್ಕೆ ಸಹಾಯ ಮಾಡಬಹುದು. ನಮ್ಮ ಸಕ್ರಿಯ ಕಾರ್ಯಕ್ರಮಗಳ ಬಗ್ಗೆ ಹೆಚ್ಚಿನ ವಿವರಗಳಿಗೆ ದಯವಿಟ್ಟು [ಮೈಕ್ರೋಸಾಫ್ಟ್ ಬಗ್ ಬೌಂಟಿ ಪ್ರೋಗ್ರಾಂ](https://aka.ms/security.md/msrc/bounty) ಪುಟವನ್ನು ಭೇಟಿ ಮಾಡಿ.

## ಆದ್ಯತೆಯ ಭಾಷೆಗಳು

ನಾವು ಎಲ್ಲಾ ಸಂವಹನಗಳನ್ನು ಇಂಗ್ಲಿಷ್‌ನಲ್ಲಿ ಇಡುವುದನ್ನು ಆದ್ಯತೆಯಾಗಿ ಪರಿಗಣಿಸುತ್ತೇವೆ.

## ನೀತಿ

ಮೈಕ್ರೋಸಾಫ್ಟ್ [ಸಮನ್ವಯಿತ ಅಪಾಯ ಬಹಿರಂಗಪಡಿಸುವ ತತ್ವ](https://aka.ms/security.md/cvd)ವನ್ನು ಅನುಸರಿಸುತ್ತದೆ.
