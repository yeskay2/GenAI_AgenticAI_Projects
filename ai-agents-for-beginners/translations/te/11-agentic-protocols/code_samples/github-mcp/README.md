# Github MCP Server ఉదాహరణ

## వివరణ

ఇది మైక్రోసాఫ్ట్ రియాక్టర్ ద్వారా నిర్వహించిన AI ఏజెంట్స్ హాకథాన్ కోసం రూపొందించిన డెమో.

ఈ టూల్ వినియోగదారుడి Github రిపోజిటరీల ఆధారంగా హాకథాన్ ప్రాజెక్టులను సిఫారసు చేయడానికి ఉపయోగించబడుతుంది. ఇది ఈ విధంగా పనిచేస్తుంది:

1. **Github ఏజెంట్** - Github MCP సర్వర్‌ను ఉపయోగించి రిపోజిటరీలు మరియు వాటి సమాచారం పొందడం.
2. **హాకథాన్ ఏజెంట్** - Github ఏజెంట్ నుండి డేటాను తీసుకుని, వినియోగదారుడు ఉపయోగించిన ప్రాజెక్టులు, భాషలు మరియు AI ఏజెంట్స్ హాకథాన్ ట్రాక్స్ ఆధారంగా సృజనాత్మక హాకథాన్ ప్రాజెక్ట్ ఆలోచనలను రూపొందించడం.
3. **ఈవెంట్స్ ఏజెంట్** - హాకథాన్ ఏజెంట్ సూచనల ఆధారంగా, AI ఏజెంట్ హాకథాన్ సిరీస్ నుండి సంబంధిత ఈవెంట్లను సిఫారసు చేస్తుంది.

## కోడ్ నడపడం

### ఎన్విరాన్‌మెంట్ వేరియబుల్స్

ఈ డెమో Azure Open AI Service, Semantic Kernel, Github MCP Server మరియు Azure AI Search ఉపయోగిస్తుంది.

ఈ టూల్స్ ఉపయోగించడానికి సరైన ఎన్విరాన్‌మెంట్ వేరియబుల్స్ సెట్ చేయడం నిర్ధారించుకోండి:

```python
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=""
AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME=""
AZURE_OPENAI_ENDPOINT=""
AZURE_OPENAI_API_KEY=""
AZURE_OPENAI_API_VERSION=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## చైన్‌లిట్ సర్వర్ నడపడం

MCP సర్వర్‌కు కనెక్ట్ అవ్వడానికి, ఈ డెమో చాట్ ఇంటర్‌ఫేస్‌గా Chainlit ఉపయోగిస్తుంది.

సర్వర్ నడపడానికి, మీ టెర్మినల్‌లో ఈ కింది కమాండ్‌ను ఉపయోగించండి:

```bash
chainlit run app.py -w
```


ఇది మీ చైన్‌లిట్ సర్వర్‌ను `localhost:8000` లో ప్రారంభించడమే కాకుండా, `event-descriptions.md` కంటెంట్‌తో మీ Azure AI Search Index ను పూరించాలి.

## MCP సర్వర్‌కు కనెక్ట్ అవ్వడం

Github MCP సర్వర్‌కు కనెక్ట్ అవ్వడానికి, "Type your message here.." చాట్ బాక్స్ కింద ఉన్న "plug" ఐకాన్‌ను ఎంచుకోండి:

![MCP Connect](../../../../../translated_images/te/mcp-chainlit-1.7ed66d648e3cfb28.webp)

అక్కడ నుండి, "Connect an MCP" పై క్లిక్ చేసి, Github MCP సర్వర్‌కు కనెక్ట్ అయ్యే కమాండ్‌ను జోడించండి:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```


"[YOUR PERSONAL ACCESS TOKEN]" ను మీ అసలు Personal Access Token తో మార్చండి.

కనెక్ట్ అయిన తర్వాత, కనెక్ట్ అయినట్లు నిర్ధారించడానికి plug ఐకాన్ పక్కన (1) కనిపిస్తుంది. కనెక్ట్ కాలేకపోతే, `chainlit run app.py -w` తో చైన్‌లిట్ సర్వర్‌ను మళ్లీ ప్రారంభించండి.

## డెమో ఉపయోగించడం

హాకథాన్ ప్రాజెక్టులను సిఫారసు చేసే ఏజెంట్ వర్క్‌ఫ్లోను ప్రారంభించడానికి, మీరు ఈ విధంగా సందేశాన్ని టైప్ చేయవచ్చు:

"Github వినియోగదారుడు koreyspace కోసం హాకథాన్ ప్రాజెక్టులను సిఫారసు చేయండి"

రౌటర్ ఏజెంట్ మీ అభ్యర్థనను విశ్లేషించి, మీ ప్రశ్నను నిర్వహించడానికి ఉత్తమమైన ఏజెంట్ల (GitHub, Hackathon, మరియు Events) కలయికను నిర్ణయిస్తుంది. ఈ ఏజెంట్లు కలిసి, Github రిపోజిటరీ విశ్లేషణ, ప్రాజెక్ట్ ఆలోచనలు మరియు సంబంధిత టెక్ ఈవెంట్ల ఆధారంగా సమగ్ర సిఫారసులను అందిస్తాయి.
