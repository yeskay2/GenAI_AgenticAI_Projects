# 🤝 Εταιρικά Συστήματα Ροής Εργασίας Πολλαπλών Πρακτόρων (.NET)

## 📋 Στόχοι Μάθησης

Αυτό το σημειωματάριο δείχνει πώς να δημιουργήσετε εξελιγμένα εταιρικά συστήματα πολλαπλών πρακτόρων χρησιμοποιώντας το Microsoft Agent Framework στο .NET με μοντέλα του GitHub. Θα μάθετε να συντονίζετε πολλούς εξειδικευμένους πράκτορες που συνεργάζονται μέσω δομημένων ροών εργασίας, αξιοποιώντας τις εταιρικές δυνατότητες του .NET για λύσεις έτοιμες για παραγωγή.

**Δυνατότητες Εταιρικών Συστημάτων Πολλαπλών Πρακτόρων που θα Δημιουργήσετε:**
- 👥 **Συνεργασία Πρακτόρων**: Συντονισμός πρακτόρων με ασφάλεια τύπου και επικύρωση κατά τη μεταγλώττιση
- 🔄 **Ορχήστρωση Ροής Εργασίας**: Δηλωτικός ορισμός ροής εργασίας με ασύγχρονα πρότυπα του .NET
- 🎭 **Εξειδίκευση Ρόλων**: Πρακτόρες με ισχυρά τυποποιημένες προσωπικότητες και τομείς εξειδίκευσης
- 🏢 **Εταιρική Ενσωμάτωση**: Πρότυπα έτοιμα για παραγωγή με παρακολούθηση και διαχείριση σφαλμάτων

## Τι Δείχνει Αυτό το Παράδειγμα

Αυτό το σύστημα ροής εργασίας πολλαπλών πρακτόρων δημιουργεί μια υπηρεσία προτάσεων ταξιδιών ξενοδοχείου με δύο εξειδικευμένους πράκτορες:

1. **Πράκτορας Υποδοχής**: Ένας ταξιδιωτικός πράκτορας που παρέχει προτάσεις δραστηριοτήτων και τοποθεσιών
2. **Πράκτορας Concierge**: Εξετάζει τις προτάσεις για να διασφαλίσει αυθεντικές, μη τουριστικές εμπειρίες

Οι πράκτορες συνεργάζονται σε μια ροή εργασίας όπου:
- Ο Πράκτορας Υποδοχής λαμβάνει το αρχικό αίτημα ταξιδιού
- Ο Πράκτορας Concierge εξετάζει και βελτιώνει την πρόταση
- Η ροή εργασίας μεταδίδει απαντήσεις σε πραγματικό χρόνο

## Βασικές Έννοιες

### Συντονισμός Πρακτόρων
Το παράδειγμα δείχνει συντονισμό πρακτόρων με ασφάλεια τύπου χρησιμοποιώντας το Microsoft Agent Framework με επικύρωση κατά τη μεταγλώττιση.

### Ορχήστρωση Ροής Εργασίας
Χρησιμοποιεί δηλωτικό ορισμό ροής εργασίας με ασύγχρονα πρότυπα του .NET για τη σύνδεση πολλών πρακτόρων σε μια αλυσίδα.

### Μεταδόσεις Απαντήσεων
Υλοποιεί μετάδοση απαντήσεων πρακτόρων σε πραγματικό χρόνο χρησιμοποιώντας ασύγχρονα enumerable και αρχιτεκτονική βασισμένη σε γεγονότα.

### Εταιρική Ενσωμάτωση
Δείχνει πρότυπα έτοιμα για παραγωγή, συμπεριλαμβανομένων:
- Ρύθμιση μεταβλητών περιβάλλοντος
- Ασφαλής διαχείριση διαπιστευτηρίων
- Διαχείριση σφαλμάτων
- Ασύγχρονη επεξεργασία γεγονότων
