## ഇവന്റിന്റെ പേര്: Azure AI Agent Service ഉപയോഗിച്ച് നിങ്ങളുടെ കോഡ്-ഫസ്റ്റ് ആപ്പ് നിർമ്മിക്കുക (EMEA/US ഓഫറിംഗ്)

## വിവരണം

Azure AI Agent Service ഒരു സേവനവും SDKയും സംയോജിപ്പിക്കുന്നതിലൂടെ ശക്തമായ AI-ചാലിതമായ പരിഹാരങ്ങൾ വികസിപ്പിക്കുന്നതിനെ ലളിതമാക്കുന്നു. ഈ സെഷനിൽ, Azure ഉപയോഗിച്ച് നിങ്ങളുടെ സ്വന്തം കോഡ്-ഫസ്റ്റ് AI ഏജന്റ് നിർമ്മിക്കുന്നതിനെക്കുറിച്ച് നിങ്ങൾ പഠിക്കും, ഇത് ചോദ്യങ്ങൾക്ക് ഉത്തരം നൽകാനും, ഡാറ്റ വിശകലനം നടത്താനും, ബാഹ്യ ഡാറ്റ സ്രോതസുകൾ സംയോജിപ്പിക്കാനും കഴിയും. കൂടാതെ, ഒന്നിച്ച് പ്രവർത്തിക്കുന്ന നിരവധി ഏജന്റുകൾ ഉൾപ്പെടുന്ന കൂടുതൽ സങ്കീർണ്ണമായ ആർക്കിടെക്ചറുകൾ നിങ്ങൾ അന്വേഷിക്കും.

## ഇവന്റിന്റെ പേര്: Semantic Kernel ഉപയോഗിച്ച് മൾട്ടി-ഏജന്റ് AI ഉപയോഗിച്ച് ബിസിനസ് പ്രക്രിയകൾ മാറ്റം വരുത്തുക

ഗ്രൂപ്പ്-ചാറ്റ്, റിഫ്ലക്ഷൻ, സെലക്ടർ, സ്വാർം എന്നിവ ഉൾപ്പെടുന്ന പാറ്റേണുകളുമായി ലൈവ് ഡെമോൺസ്ട്രേഷനുകളും പ്രായോഗിക പഠനവും വഴി മൾട്ടി-ഏജന്റ് AI സിസ്റ്റങ്ങളുടെ ശക്തി കണ്ടെത്തുക. Python ഉപയോഗിച്ച് ഉപഭോക്തൃ പിന്തുണ മുതൽ പ്രോജക്റ്റ് മാനേജ്മെന്റ് വരെ നിർണായകമായ ബിസിനസ് പ്രക്രിയകൾ ഓട്ടോമേറ്റ് ചെയ്യാനും സ്കെയിൽ ചെയ്യാനും Semantic Kernel Process Framework ഉപയോഗിക്കുക.

## വിവരണം

GitHub Models-ന്റെ സഹായത്തോടെ, നിങ്ങളുടെ ആദ്യ AI ഏജന്റ് നിർമ്മിക്കാൻ GitHub അക്കൗണ്ട് മാത്രം മതിയാകും! GitHub Models-ൽ OpenAI gpt-4o, DeepSeek-R1, Llama-3.1 എന്നിവ പോലുള്ള ശക്തമായ മോഡലുകൾ ഉൾപ്പെടുന്നു, പ്ലേഗ്രൗണ്ടിൽ അല്ലെങ്കിൽ നിങ്ങളുടെ കോഡിൽ പരീക്ഷിക്കാൻ തയ്യാറാണ്. Python ഉപയോഗിച്ച് GitHub Models-ലേക്ക് എങ്ങനെ കണക്റ്റ് ചെയ്യാമെന്ന് ഈ സെഷനിൽ ഞങ്ങൾ കാണിക്കും, തുടർന്ന് PydanticAI, AutoGen, Semantic Kernel പോലുള്ള പ്രശസ്ത Python പാക്കേജുകൾ ഉപയോഗിച്ച് ഏജന്റുകൾ നിർമ്മിക്കുക. GitHub Codespaces-ൽ ലൈവായി പിന്തുടരാം, അല്ലെങ്കിൽ സെഷനിന് ശേഷം നിങ്ങൾക്ക് സ്വയം ഉദാഹരണങ്ങൾ പരീക്ഷിക്കാം.

## ഇവന്റിന്റെ പേര്: Azure AI മോഡൽ കാറ്റലോഗിൽ നിന്ന് മോഡലുകളുടെ സൈന്യവുമായി ഏജന്റുകൾ നിർമ്മിക്കുക

## വിവരണം

Azure AI മോഡൽ കാറ്റലോഗ് വിവിധ കഴിവുകളും കഴിവുകളും ഉള്ള മോഡലുകളുടെ വലിയ വൈവിധ്യം നൽകുന്നു. ആരംഭിക്കാൻ ഒരു റെഡി-ടു-യൂസ് മോഡൽ ഉപയോഗിക്കുമ്പോൾ, ഡെവലപ്പർമാർ കൂടുതൽ സങ്കീർണ്ണമായ വർക്ക്‌ഫ്ലോകൾ ഉപയോഗിക്കുമ്പോൾ, അവരുടെ ഇഷ്ടമായ ഫ്രെയിംവർക്കിൽ ജോലികൾ ചെയ്യാൻ പ്രത്യേക മോഡലുകൾ ഉപയോഗിക്കാം. മോഡൽ കാറ്റലോഗ് ഓഫറിംഗ്, മോഡലുകളുടെ സൈന്യത്തിന്റെ മുകളിൽ ഇരിക്കുന്ന ഏജന്റുകൾ എങ്ങനെ നിർമ്മിക്കാമെന്ന് ഈ പ്രദർശനത്തിൽ ഞങ്ങൾ വിശദീകരിക്കുന്നു - കൂടാതെ ഇത് നിങ്ങൾക്ക് വലിയ ചെലവാകാതെ.

## ഇവന്റിന്റെ പേര്: LangGraph, Azure Cosmos DB ഉപയോഗിച്ച് മൾട്ടി-ഏജന്റ് API

## വിവരണം

മൾട്ടി-ഏജന്റ് AI അപ്ലിക്കേഷനുകളുടെ ഉയർച്ച ബുദ്ധിമാനായ സിസ്റ്റങ്ങൾ നിർമ്മിക്കുന്നതിൽ മാറ്റം വരുത്തുന്നു - എന്നാൽ അവയെ യഥാർത്ഥ ലോക സ്കെയിലബിലിറ്റിക്കും പ്രകടനത്തിനും വേണ്ടി എങ്ങനെ ആർക്കിടെക്ചർ ചെയ്യാം? LangGraph ഉപയോഗിച്ച് ഏജന്റ് ഓർക്കസ്ട്രേഷൻ, API ലെയറിനായി FastAPI, സ്റ്റേറ്റ് മാനേജ്മെന്റിനും വെക്ടർ സ്റ്റോറേജിനും ട്രാൻസാക്ഷണൽ ഡാറ്റയ്ക്കും ബാക്ക്ബോൺ ആയി Azure Cosmos DB ഉപയോഗിച്ച് നിർമ്മിച്ച പ്രൊഡക്ഷൻ-ഗ്രേഡ് മൾട്ടി-ഏജന്റ് അപ്ലിക്കേഷൻ ഞങ്ങൾ വിശദമായി പരിശോധിക്കും.

വിവര-അമർത്തിയ ജനന (RAG), മെമ്മറി സ്ഥിരത, ഡൈനാമിക് സ്റ്റേറ്റ് ട്രാൻസിഷൻ എന്നിവ സംയോജിപ്പിക്കുന്ന ഏജന്റ്-ചാലിതമായ വർക്ക്‌ഫ്ലോ എങ്ങനെ രൂപകൽപ്പന ചെയ്യാമെന്നും നടപ്പിലാക്കാമെന്നും ഒരു വിശദമായ കോഡ് വാക്ക്‌ത്രൂ വഴി നിങ്ങൾ കാണും. ഞങ്ങൾ ഉൾപ്പെടുത്തും:

LangGraph ഉപയോഗിച്ച് ഘടനാപരമായ ചിന്തയ്ക്കുള്ള ഏജന്റ് സഹകരണം  
Azure Cosmos DB ഉപയോഗിച്ച് റിയൽ-ടൈം ചാറ്റ് ചരിത്ര സംഭരണം - ChatGPT, ചരിത്രത്തിലെ ഏറ്റവും വേഗത്തിൽ വളരുന്ന AI ഏജന്റ് അപ്ലിക്കേഷൻ, പവർ ചെയ്യുന്ന അതേ ഡാറ്റാബേസ്  
Cosmos DB-യുടെ നാടൻ എംബെഡിംഗ്സ് പിന്തുണയുള്ള അറിവ് തിരച്ചിലിനായി വെക്ടർ തിരയൽ  
FastAPI-യുടെ അസിങ്ക് കഴിവുകൾ ഇടപെടലുകൾ പ്രതികരണശീലവും സ്കെയിലബിളും ആക്കാൻ
