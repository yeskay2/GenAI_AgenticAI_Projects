# Github MCP Server ഉദാഹരണം

## വിവരണം

മൈക്രോസോഫ്റ്റ് റിയാക്ടർ വഴി സംഘടിപ്പിച്ച AI ഏജന്റ്സ് ഹാക്കത്തോൺക്കായി സൃഷ്ടിച്ച ഒരു ഡെമോയാണ് ഇത്.

ഉപയോക്താവിന്റെ Github റിപോസിറ്ററികൾ അടിസ്ഥാനമാക്കി ഹാക്കത്തോൺ പ്രോജക്റ്റുകൾ ശുപാർശ ചെയ്യാൻ ഈ ടൂൾ ഉപയോഗിക്കുന്നു. ഇത് ചുവടെയുള്ള രീതിയിൽ പ്രവർത്തിക്കുന്നു:

1. **Github ഏജന്റ്** - Github MCP സെർവർ ഉപയോഗിച്ച് റിപോസിറ്ററികളും അവയുടെ വിവരങ്ങളും തിരികെ ലഭിക്കുന്നു.
2. **ഹാക്കത്തോൺ ഏജന്റ്** - Github ഏജന്റിൽ നിന്ന് ലഭിക്കുന്ന ഡാറ്റ ഉപയോഗിച്ച്, ഉപയോക്താവിന്റെ പ്രോജക്റ്റുകൾ, ഉപയോഗിക്കുന്ന ഭാഷകൾ, AI ഏജന്റ്സ് ഹാക്കത്തോൺ ട്രാക്കുകൾ എന്നിവയെ അടിസ്ഥാനമാക്കി സൃഷ്ടിപരമായ ഹാക്കത്തോൺ പ്രോജക്റ്റ് ആശയങ്ങൾ നിർദ്ദേശിക്കുന്നു.
3. **ഇവന്റ്സ് ഏജന്റ്** - ഹാക്കത്തോൺ ഏജന്റിന്റെ ശുപാർശയെ അടിസ്ഥാനമാക്കി, AI ഏജന്റ്സ് ഹാക്കത്തോൺ സീരീസിലെ അനുയോജ്യമായ ഇവന്റുകൾ ശുപാർശ ചെയ്യുന്നു.

## കോഡ് പ്രവർത്തിപ്പിക്കൽ

### പരിസ്ഥിതി ചോദ്യങ്ങൾ

ഈ ഡെമോ Azure Open AI Service, Semantic Kernel, Github MCP Server, Azure AI Search എന്നിവ ഉപയോഗിക്കുന്നു.

ഈ ടൂളുകൾ ഉപയോഗിക്കാൻ ആവശ്യമായ പരിസ്ഥിതി ചോദ്യങ്ങൾ സജ്ജമാക്കുക:

```python
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=""
AZURE_OPENAI_EMBEDDING_DEPLOYMENT_NAME=""
AZURE_OPENAI_ENDPOINT=""
AZURE_OPENAI_API_KEY=""
AZURE_OPENAI_API_VERSION=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## Chainlit സെർവർ പ്രവർത്തിപ്പിക്കൽ

MCP സെർവറുമായി ബന്ധപ്പെടാൻ, ഈ ഡെമോ Chainlit എന്ന ചാറ്റ് ഇന്റർഫേസ് ഉപയോഗിക്കുന്നു.

സെർവർ പ്രവർത്തിപ്പിക്കാൻ, നിങ്ങളുടെ ടെർമിനലിൽ താഴെ കാണുന്ന കമാൻഡ് ഉപയോഗിക്കുക:

```bash
chainlit run app.py -w
```


ഇത് നിങ്ങളുടെ Chainlit സെർവർ `localhost:8000`-ൽ ആരംഭിക്കുകയും `event-descriptions.md` ഉള്ളടക്കത്തോടെ Azure AI Search Index പൂരിപ്പിക്കുകയും ചെയ്യും.

## MCP സെർവറുമായി ബന്ധപ്പെടൽ

Github MCP സെർവറുമായി ബന്ധപ്പെടാൻ, "Type your message here.." ചാറ്റ് ബോക്സിന് താഴെയുള്ള "plug" ഐക്കൺ തിരഞ്ഞെടുക്കുക:

![MCP Connect](../../../../../translated_images/ml/mcp-chainlit-1.7ed66d648e3cfb28.webp)

അവിടെ നിന്ന് "Connect an MCP" ക്ലിക്ക് ചെയ്ത് Github MCP സെർവറുമായി ബന്ധപ്പെടാനുള്ള കമാൻഡ് ചേർക്കുക:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```


"[YOUR PERSONAL ACCESS TOKEN]" നിങ്ങളുടെ യഥാർത്ഥ Personal Access Token-നാൽ മാറ്റുക.

ബന്ധപ്പെടുന്നതിന് ശേഷം, plug ഐക്കണിന് സമീപം (1) കാണുന്നതിലൂടെ ഇത് ബന്ധപ്പെട്ടതായി സ്ഥിരീകരിക്കാം. അല്ലെങ്കിൽ, `chainlit run app.py -w` ഉപയോഗിച്ച് chainlit സെർവർ വീണ്ടും ആരംഭിക്കാൻ ശ്രമിക്കുക.

## ഡെമോ ഉപയോഗിക്കൽ

ഹാക്കത്തോൺ പ്രോജക്റ്റുകൾ ശുപാർശ ചെയ്യാനുള്ള ഏജന്റ് വർക്ക്‌ഫ്ലോ ആരംഭിക്കാൻ, നിങ്ങൾക്ക് ഇങ്ങനെ ഒരു സന്ദേശം ടൈപ്പ് ചെയ്യാം:

"Github ഉപയോക്താവ് koreyspace-ക്കായി ഹാക്കത്തോൺ പ്രോജക്റ്റുകൾ ശുപാർശ ചെയ്യുക"

Router Agent നിങ്ങളുടെ അഭ്യർത്ഥന വിശകലനം ചെയ്ത്, ഏത് ഏജന്റുകളുടെ (GitHub, Hackathon, Events) സംയോജനം നിങ്ങളുടെ ചോദ്യം കൈകാര്യം ചെയ്യാൻ ഏറ്റവും അനുയോജ്യമാണെന്ന് നിർണ്ണയിക്കും. Github റിപോസിറ്ററി വിശകലനം, പ്രോജക്റ്റ് ആശയങ്ങൾ, അനുയോജ്യമായ ടെക് ഇവന്റുകൾ എന്നിവയെ അടിസ്ഥാനമാക്കി ഏജന്റുകൾ ഒരുമിച്ച് പ്രവർത്തിച്ച് സമഗ്രമായ ശുപാർശകൾ നൽകും.
